In this application you can see some graphical results on the kaggle database "Bank-Churnes". The variables involved in the analyses are the following, but we will focus on the analyses of the variable Income_category.
  • Attrition_Flag: Internal event (customer activity) variable - if the account is closed then 1 else 0
  • Customer_Age: Demographic variable - Customer's Age in Years
  • Education_Level: Demographic variable - Educational Qualification of the account holder (example: high school, college graduate, etc.).
  • Card-Category: Product Variable - Type of Card (Blue, Silver, Gold, Platinum).
  • Credit_Limit: Credit Limit on the Credit Card.
  • Income_Category:Demographic variable - Annual Income Category of the account holder
  • Total_Trans_Amt: Total Transaction Amount (Last 12 months)
  • Total_Trans_Ct: Total Transaction Count (Last 12 months).
Finally, you will find the following information panels:
  • Summary tab:You can obtain summary information on both numerical and categorical variables. You can also view the data with any number of observations.
  • R plots tab: You will be able to make boxplots and hisgrams for the numerical variables of the dataset. You will also be able to observe boxplots as a function of categorical variables.
  • Plotly tab: you will observe dynamic graphs thanks to the function useshinyjs where you will obtain information about the represented observations.
  • Machine Learning tab:in this panel you will find a summary of two machine learning projects, in the first subtab you can obtain the optimal number of clusters for the data, then you will have a subtab of the most important variables thanks to a training model with the caret library.
